ExtremeII OS by Daogu.

  1. bilibili视频搜索
    • 支持搜索视频并在PlayX上播放
    • 更多的虫子
  2. 网页浏览器
    • 搜索
      • 必应
        • 网页搜索
          • 中国版、国际版切换
        • 图片搜索
          • 中国版、国际版切换
      • 百度
        • 网页搜索
        • 图片搜索
  3. 网易云音乐
    • 支持搜索非VIP音乐并在Stream Radio播放
    • 带有时间条
    • 带有播放控件
    • 修复了虫子
